# Onboarding Checklist

Use this checklist when a new user installs the skill or when account context is incomplete.

## 1. Access and account config

Collect:

- Google Ads customer ID
- Optional MCC / login customer ID
- Developer token source
- OAuth method / refresh token path / service auth path
- Whether existing auth is already configured on the machine
- Reporting timezone
- Preferred default date ranges (7d / 14d / 30d)

## 2. Business context

Collect:

- Product/service name
- Business model (SaaS, e-commerce, lead gen, local service, agency, app, etc.)
- Core offer / pricing structure
- Primary countries and languages
- Main competitors
- Landing pages or domains in use

## 3. Measurement and KPIs

Collect:

- Primary conversion action name and ID
- Secondary conversion actions, if any
- Whether optimization should use Purchase, lead, signup, booked call, trial, etc.
- Revenue/value source
- Target CPA / target ROAS / target CAC / payback rule
- Attribution caveats or reporting caveats

## 4. Campaign structure context

Collect:

- Active campaign types (Search / PMax / Demand Gen / Display / Video / App)
- Naming conventions
- Existing brand / competitor / generic / remarketing segmentation
- Geographic segmentation rules
- Device segmentation rules
- Budget constraints
- Bidding strategy rules

## 5. Keywords and messaging inputs

Collect:

- Core keyword buckets
- Match type preferences
- Negative keyword principles
- High-intent vs exploratory keyword definitions
- Product positioning / key benefits
- Claims or compliance restrictions
- CTA preferences

## 6. Write permissions and safety

Before creating or editing anything, confirm:

- Whether reads only or writes are allowed
- Whether new objects must stay PAUSED by default
- Maximum allowed daily budget for tests
- Whether ads should be created now or only campaign skeletons
- Whether tracking templates / final URL suffix / UTM rules are required

## 7. Deliverables to offer after onboarding

After collecting the above, offer one of these next steps:

- account audit
- low CTR diagnosis
- Purchase keyword analysis
- search term cleanup and negatives
- paused campaign skeleton build
- campaign structure redesign
